1.vi和vim

vim是一个文本编辑器，它具有三种模式：

可以用来写代码：

首先输入：vi Hello.java 或者 vim Hello.java (使用vim编辑器编写java程序) 回车后进入正常模式 之后输入i等字母，进入插入模式(编辑模式) 然后写代码

1.正常模式：

以vim打开一个文档就是默认模式。可以使用上下左右按键来移动光标。

快捷键 模式 介绍
yy 正常模式 复制当前行
p 正常模式 粘贴
5yy 正常模式 复制当前行向下的5行
dd 正常模式 删除当前行
5dd 正常模式 删除当前行向下的5行

快捷键到该文档的最首行(正常模式)：gg

快捷键到该文档的最末行(正常模式)：G

撤销动作(比如删除某一行，发现删错了，然后恢复这一行)(正常模式)：u

在某个文件中，想让光标移动到第20行(正常模式)：输入20，再输入 shift+g

2.插入模式：

按下i,I,o,O,a,A,r,R任意一个字母进入编辑模式 。

3.命令行模式:

输入要查找的单词，回车查找 输入n查找到下一个

给文件每行设置行号： 先冒号: 然后set nu (set number) 回车

取消行号： 先冒号: 然后set nonu 回车

2.开机、重启和用户登录注销

命令 介绍
shutdown -h now 立即关机
halt 立即关机
shutdown 1分钟后关机
shutdown -h 1 给每一个登录到Linux系统的用户发一条消息：1分钟后关机
shutdown -h 2 2分钟后关机
shutdown -r now 立刻重启
reboot 立刻重启
sync 把内存数据同步到磁盘
注意 不管是重启还是关机，首先要运行 sync 命令，把内存中的数据写到磁盘中；目前的 shutdown/reboot/halt 等命令均已经在关机前进行了sync，但还是手动执行一下sync保险
1.用户的登录和注销

用户登录:登录时尽量少用 root 帐号登录，因为它是系统管理员，最大的权限，避免操作失误。 而且以后也用不到root，除非你是系统管理员。

可以利用普通用户登录，登录后再用”**su - 用户名**’命令来切换成系统管理员身份.

语法logout:注销用户：在图形运行级别无效

3.用户管理

一个Linux系统中只有一个root用户，但是由root用户可以创建多个其他用户，可以给这些用户分配权限， 其中每个用户会对应一个目录，这个目录叫做家目录。

1.添加用户

语法 useradd 用户名*

例如：useradd sb 创建一个叫sb的新用户

创建成功后会自动生成和新用户同名的文件夹(家目录),在home目录下

也可以指定目录：useradd -d 指定目录 新用户名

例如：useradd -d /home/fw sb 新建名叫sb的用户，家目录为fw

2.设置/修改密码

语法：passwd 用户名(不写用户名默认修改当前用户密码)

命令：pwd:显示当前用户所在目录 Print Working Directory

3.删除用户

语法：userdel 用户名(只有root用户有删除权限且不能删除自己)

删除用户，保留家目录：

userdel 用户名

一同删除：userdel -r 用户名

4.查询用户信息

语法：id 用户名

id root: uid=0(root) gid=0(root) 组=0(root)

userid是0 gruopid组id是0 所在组0

当用户不存在：返回无此用户 no such user

5.切换用户

语法：su -用户名

注意： 1.从权限高的用户切换到权限低的用户，不需要输入密码，反之需要。 2.当需要返回到原来用户时，使用 exit/logout 指令

6.查看当前用户信息

语法：who am i

假设A用户登录系统，切换到B用户，B输入命令who am i，输出的还是A用户的信息
